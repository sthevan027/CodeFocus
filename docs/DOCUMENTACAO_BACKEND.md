# 🔧 Documentação Back-end – CodeFocus

## Git Integration
- Detecta se há repositório com `simple-git`
- Ao fim do ciclo, verifica decisão do usuário
- Se continuar codando: executa `git add . && git commit`

## Histórico
- Salva nome, duração e horário do ciclo em `history.json`

## Relatórios
- Ao fim do dia, `report-generator.js` monta `.txt` com resumo
