# 🎨 Documentação Front-end – CodeFocus

## Estrutura dos Componentes

- `TimerDisplay.jsx` → contador visual com progresso
- `CycleModal.jsx` → define nome do ciclo Pomodoro
- `CommitPrompt.jsx` → janela de decisão para commit
- `Header.jsx` → barra de título com tema e status

## Estilo

- TailwindCSS com design em glassmorphism
- Suporte a dark/light mode
- Responsivo para notebooks e desktops
