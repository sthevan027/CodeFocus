## 🚀 **Como Executar**

### 1. **Backend**
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. **Frontend**
```bash
npm install
npm start
```
